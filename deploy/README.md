# 排课结果检查器

一个用于检查学校排课结果合理性的在线工具。

## 功能特点

- 检查每班每科目是否重复
- 检查老师时间冲突
- 检查教室使用冲突
- 支持艺术体育课程识别
- 可视化冲突高亮显示

## 使用方法

1. 访问网站
2. 上传或拖拽Excel排课文件
3. 查看检查结果和可视化表格

## 支持的文件格式

- Excel文件(.xlsx, .xls)
- 按星期一到星期五分组的排课表

## 技术栈

- 纯前端HTML/CSS/JavaScript
- SheetJS库用于Excel文件解析
- 无需后端服务器

## 部署

本项目已部署到GitHub Pages，可直接在线使用。

## 本地使用

直接打开`course_checker.html`文件即可在本地使用。
